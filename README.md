# 🎨 Midas Carrossel - Gerador de Carrosseis com IA\n\n> Ferramenta inteligente para criar carrosseis virais para Instagram e redes sociais usando IA Claude.\n\n## 🚀 Fase 1 - Core Features Implementadas\n\nEsta versão implementa todas as funcionalidades essenciais para criação profissional de carrosseis:\n\n### ✅ 1. Sistema de Nichos\n- Tela inicial que pergunta qual é seu nicho (Instagram, E-commerce, Blog, Educação, Saúde, etc)\n- Armazenamento em localStorage\n- Templates e sugestões de IA personalizadas por nicho\n\n### ✅ 2. Editor Visual Completo (WYSIWYG)\n- **Preview em tempo real** dos slides\n- **Clique para editar** qualquer elemento (texto, imagem)\n- **Painel lateral de edição** com opções:\n  - Editar texto com IA helper\n  - Alterar cor do texto (color picker)\n  - Alterar fonte (Arial, Georgia, Playfair Display, Montserrat, etc)\n  - Alterar tamanho do texto (slider 12px a 72px)\n  - Deletar elemento\n- **Mudanças em tempo real** no slide\n\n### ✅ 3. Gerenciamento de Imagens\n- Upload com preview\n- Drag & drop para reposicionar nos slides\n- Deletar imagens com botão dedicado\n- Reordenar imagens (suporte a drag)\n- Corrigir deformação com `object-fit: cover`\n- Feedback visual ao reordenar\n\n### ✅ 4. Integração IA Melhorada\n- Botão \"🎯 Gerar com IA\" que pergunta:\n  - Qual é o assunto/ideia principal?\n  - Qual tom? (Profissional, Casual, Urgente, etc)\n  - Quantos slides?\n- **Claude Haiku API** para gerar textos baseado NO NICHO\n- Opção de \"Regenerar\" um slide individual com IA\n- Sugestões inteligentes baseadas no contexto\n\n### ✅ 5. Sistema de CTA Melhorado\n- Slide CTA dedicado com:\n  - Texto customizável\n  - Opção de posicionar (início, meio, fim)\n  - Texto e link customizáveis\n- Slide \"Like\" para engagement\n- Diferentes CTAs por objetivo:\n  - Salvamentos (Save, Pinterest)\n  - Comentários (Comment, Engage)\n  - Encaminhamentos (Share, Forward)\n  - Compras (Buy, Link)\n  - Inscrições (Subscribe)\n\n### ✅ 6. Sistema de Favoritos\n- Botão \"⭐ Favorito\" para salvar carrossel\n- Armazenamento completo em localStorage:\n  - ID único\n  - Nome customizável\n  - Data de criação\n  - Thumbnail dos slides\n- Aba \"Favoritos\" para gerenciar salvos\n- Clique para abrir e editar novamente\n\n### ✅ 7. Sistema de Projetos\n- Aba \"📁 Projetos\" que lista:\n  - Todos os carrousséis criados\n  - Último acesso\n  - Opção de abrir (carregar completo)\n  - Opção de deletar\n  - Opção de duplicar\n- **Auto-save** a cada mudança\n- Salvar com nome customizável\n\n### ✅ 8. Histórico\n- Aba \"📋 Histórico\" mostrando:\n  - Últimas gerações com IA (até 50)\n  - Quando foram criadas\n  - Botão para usar como template\n  - Tipo de geração\n\n### ✅ 9. Exportação Múltiplos Formatos\n- **PNG** (com transparência, alta qualidade)\n- **JPG** (alta qualidade, fundo branco)\n- **ZIP** (todos os slides em um arquivo)\n- Download individual de slides\n- Download em lote com espera entre downloads\n- **Corrigido:** Bug de deformação de imagens na exportação\n\n### ✅ 10. Melhorias de Design/UX\n- ✅ Layout responsivo (desktop + mobile)\n- ✅ Aumentado tamanho de textos pequenos\n- ✅ Distribuição melhorada de elementos\n- ✅ Cores atraentes com tema escuro\n- ✅ Navegação entre abas intuitiva\n- ✅ Ícones para cada aba\n- ✅ Feedback visual completo\n\n## 📁 Estrutura de Arquivos\n\n```\nmidas-carrossel/\n├── index.html                 # HTML principal (refatorado)\n├── css/\n│   ├── style.css             # Estilos gerais\n│   ├── editor.css            # Editor visual\n│   └── responsive.css        # Responsividade\n├── js/\n│   ├── main.js              # Lógica principal\n│   ├── editor.js            # Editor WYSIWYG\n│   ├── ai-integration.js    # Integração Claude\n│   ├── storage.js           # localStorage (favoritos, projetos, histórico)\n│   ├── export.js            # Exportação múltiplos formatos\n│   └── utils.js             # Funções utilitárias\n├── api/\n│   └── gerar.js             # Serverless Vercel (proxy Claude)\n├── README.md                # Este arquivo\n└── .git/                    # Repositório Git\n```\n\n## 🛠️ Tecnologias Utilizadas\n\n- **Frontend:** HTML5, CSS3, Vanilla JavaScript\n- **AI:** Claude Haiku 4.5 API (via Vercel Serverless)\n- **Storage:** localStorage (persistência local)\n- **Export:** html2canvas, jszip\n- **Design:** CSS Grid, Flexbox, Variáveis CSS\n\n## 🎯 Como Usar\n\n### 1. Selecionar Nicho\nNa primeira vez, você verá uma tela pedindo seu nicho. Escolha entre as opções ou digite um personalizado.\n\n### 2. Criar Carrossel\n1. Digite o **tema** do carrossel\n2. Clique em \"✦ Gerar ideias\" para sugestões de IA\n3. Configure:\n   - **Público-alvo**\n   - **Tom** (Profissional, Casual, Urgente, etc)\n   - **Handle** da marca\n   - **Número de slides**\n   - **Slides opcionais** (Capa, CTA, Like)\n4. Escolha o **Formato** (1:1, 4:5, 9:16, 16:9)\n5. Selecione o **Fundo** (cor ou imagens)\n6. Clique em \"⚡ Gerar Carrossel\"\n\n### 3. Editar Slides\n1. Clique em qualquer **elemento** (texto, imagem) para editar\n2. Use o **painel lateral** para:\n   - Alterar texto\n   - Mudar cor\n   - Ajustar tamanho\n   - Escolher fonte\n   - Deletar elemento\n3. As mudanças aparecem **em tempo real**\n\n### 4. Gerenciar Imagens\n1. Na seção **\"Fundo\"**, selecione **\"📁 Imagens\"**\n2. Clique para fazer upload\n3. Reordene com drag & drop\n4. Clique no **X** para deletar\n\n### 5. Salvar Favorito\n1. Clique em **\"⭐ Favorito\"**\n2. Digite um nome para o favorito\n3. Acesse em **\"Favoritos\"** para carregar depois\n\n### 6. Criar Projeto\n1. Clique em **\"📁 Projeto\"**\n2. Digite um nome\n3. O projeto é **auto-salvado** a cada mudança\n4. Acesse em **\"Projetos\"** para continuar editando\n\n### 7. Exportar\n- **PNG Individual:** Clique em \"⬇ PNG\" no slide\n- **Todos PNG:** Clique em \"⬇ Todos PNG\"\n- **ZIP:** Clique em \"⬇ ZIP\" (contém todos os slides)\n\n## 💾 Persistência de Dados\n\nTodos os dados são salvos em **localStorage** do navegador:\n\n- **Favoritos:** `midas-favoritos`\n- **Projetos:** `midas-projetos`\n- **Histórico:** `midas-historico`\n- **Nicho:** `midas-nicho`\n- **Estado:** `midas-state`\n\n⚠️ **Nota:** Limpar cache do navegador apagará todos os dados. Sempre exporte seus carrosseis!\n\n## 🤖 Integração com IA\n\nO sistema usa **Claude Haiku 4.5** para:\n\n1. **Gerar temas** baseados no nicho\n2. **Criar carrosseis** completos com:\n   - Capa com título destaque\n   - Slides de conteúdo numerados\n   - Slides de engajamento (Like)\n   - Call-to-Action (CTA)\n3. **Sugerir textos** para elementos\n4. **Regenerar slides** individuais\n\nA IA leva em conta:\n- Nicho do usuário\n- Tom desejado\n- Público-alvo\n- Objetivos da campanha\n\n## 📱 Responsividade\n\n- ✅ **Desktop:** Layout otimizado com 2 colunas\n- ✅ **Tablet:** Layout adaptado\n- ✅ **Mobile:** Interface completa em uma coluna\n- ✅ **Breakpoints:** 1400px, 768px, 480px\n\n## 🐛 Correções de Bugs\n\n- ✅ Deformação de imagens na exportação (fixed)\n- ✅ Memory leak de ObjectURLs\n- ✅ XSS em contenteditable (sanitizado)\n- ✅ Distribuição de elementos nos slides\n- ✅ Tamanho de fontes em mobile\n\n## 🚀 Próximas Melhorias (Fase 2)\n\n- [ ] Geração de fundos com IA\n- [ ] PDF com múltiplas páginas\n- [ ] Undo/Redo completo\n- [ ] Compartilhamento de carrosseis\n- [ ] Templates pré-prontos\n- [ ] Análise de performance de carrosseis\n- [ ] Integração com canva\n- [ ] Download direto no Instagram\n\n## 📝 Requisitos Técnicos\n\n### Frontend\n- Navegador moderno (Chrome, Firefox, Safari, Edge)\n- JavaScript habilitado\n- localStorage disponível\n\n### Backend\n- Vercel Serverless (api/gerar.js)\n- Variável de ambiente: `ANTHROPIC_API_KEY`\n- Acesso à Claude API\n\n## 🔐 Segurança\n\n- ✅ Sanitização de conteúdo edit\u00e1vel\n- ✅ Validação de entrada\n- ✅ CORS habilitado\n- ⚠️ API Key em variável de ambiente (servidor)\n- ⚠️ Sem armazenamento de dados sensíveis no cliente\n\n## 📄 Licença\n\nMIT License - Sinta-se à vontade para usar e modificar!\n\n## 👨‍💻 Contribuições\n\nEste projeto foi desenvolvido com ❤️ para criadores de conteúdo.\n\nTem ideias?
